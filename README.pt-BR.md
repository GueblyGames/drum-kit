# Bateria Virtual — Guebly Games

> Bateria virtual interativa com visualizador de áudio e compositor.

## Jogar

```bash
npm install
npm run dev
```

## Funcionalidades

- 9 pads de bateria (Bumbo, Caixa, Hi-Hat, Tons, Crash, Ride)
- Entrada por teclado e mouse/toque
- Visualizador de frequência em tempo real
- Compositor: escreva sequências e reproduza em BPM customizado
- Animação ring-out ao tocar
- Web Audio API para baixa latência
- Design responsivo
- Tema dark Guebly Games

## Tecnologias

- React 18
- Vite
- styled-components
- Web Audio API (AudioContext + AnalyserNode)
- Google Fonts (Bebas Neue, Orbitron, Rajdhani, Space Mono)

## Build

```bash
npm run build
```
