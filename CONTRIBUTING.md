# Contributing

1. Fork this repository
2. Create a branch: `git checkout -b my-feature`
3. `npm install && npm run dev` to test
4. Commit: `git commit -m 'feat: description'`
5. Push and open a Pull Request

Code style: React functional components, styled-components, ES6+.

This game is part of the Guebly Games web games collection — [games.guebly.com.br](https://games.guebly.com.br)
